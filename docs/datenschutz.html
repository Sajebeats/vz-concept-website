1. Datenschutz auf einen Blick

Allgemeine Hinweise

Die folgenden Hinweise geben einen einfachen Überblick darüber, was mit Ihren personenbezogenen Daten passiert, wenn Sie diese Website besuchen. Personenbezogene Daten sind alle Daten, mit denen Sie persönlich identifiziert werden können. Ausführliche Informationen zum Thema Datenschutz entnehmen Sie unserer unter diesem Text aufgeführten Datenschutzerklärung.

Datenerfassung auf dieser Website

Wer ist verantwortlich für die Datenerfassung auf dieser Website?
Die Datenverarbeitung auf dieser Website erfolgt durch den Websitebetreiber. Dessen Kontaktdaten können Sie dem Impressum dieser Website entnehmen.

Wie erfassen wir Ihre Daten?
Ihre Daten werden zum einen dadurch erhoben, dass Sie uns diese mitteilen – etwa durch das Buchen eines Termins über Calendly oder durch eine E-Mail-Kontaktaufnahme. Andere Daten werden automatisch oder nach Ihrer Einwilligung beim Besuch der Website durch unsere IT-Systeme erfasst (z. B. Browserdaten).

Wofür nutzen wir Ihre Daten?
Ein Teil der Daten wird erhoben, um eine fehlerfreie Bereitstellung der Website zu gewährleisten. Andere Daten können zur Analyse Ihres Nutzerverhaltens oder zur Bearbeitung Ihrer Anfragen genutzt werden.

2. Hosting

Diese Website wird über GitHub Pages bereitgestellt (Anbieter: GitHub Inc., 88 Colin P. Kelly Jr. Street, San Francisco, CA 94107, USA). Beim Aufruf der Website verarbeitet GitHub technisch notwendige Daten in Server-Logfiles, insbesondere IP-Adresse, Browser-Typ, Betriebssystem, Referrer-URL, Hostname und Datum/Uhrzeit der Serveranfrage.

Da GitHub seinen Sitz in den USA hat, kann eine Übermittlung personenbezogener Daten in die USA stattfinden. GitHub ist nach dem EU-US Data Privacy Framework zertifiziert und bietet damit ein angemessenes Datenschutzniveau (Art. 45 DSGVO). Rechtsgrundlage für die Verarbeitung ist Art. 6 Abs. 1 lit. f DSGVO (berechtigtes Interesse an einer technisch fehlerfreien Bereitstellung der Website).

Datenschutzerklärung von GitHub: github.com/de/site-policy/privacy-policies

4. Datenerfassung auf dieser Website

Schriftarten (Self-Hosted)

Diese Website nutzt die Schriftarten Inter, Playfair Display und Dancing Script zur einheitlichen Darstellung von Texten. Die Schriften werden direkt vom Server dieser Website ausgeliefert (Self-Hosting). Es findet keine Verbindung zu Google Fonts oder anderen externen Schriftanbietern statt. Beim Aufruf einer Seite werden somit keine Schriftdaten an Drittanbieter übertragen.

Calendly (Terminbuchung)

Für die Buchung von Terminen nutzen wir den Dienst Calendly (Calendly LLC, 271 17th St NW, Atlanta, GA 30363, USA). Calendly wird nicht passiv auf der Website eingebunden — die Calendly-Seite öffnet sich erst, wenn Sie aktiv den Button „Erstgespräch buchen" anklicken und auf die Calendly-Domain weitergeleitet werden.

Wenn Sie über Calendly einen Termin buchen, werden die von Ihnen eingegebenen Daten (Name, E-Mail-Adresse, optional weitere Angaben) an Calendly übertragen und dort verarbeitet. 5. Cookies und Einwilligungs-Management

Aktueller Stand

Diese Website setzt aktuell keine eigenen Tracking-Cookies und keine Marketing-Tools ein. Beim ersten Besuch erscheint ein Cookie-Banner, der dich über die geplante Nutzung von Statistik- und Marketing-Diensten informiert und deine Einwilligung einholt.

Kategorien

  • Notwendig: Speicherung deiner Cookie-Einstellungen im lokalen Browser-Speicher (localStorage). Diese Speicherung ist technisch erforderlich, damit der Banner nicht bei jedem Besuch erneut erscheint. Rechtsgrundlage: § 25 Abs. 2 TTDSG (technisch zwingend erforderlich).
  • Statistik (Opt-in): Werkzeuge zur anonymisierten Reichweitenmessung (z. B. Google Analytics 4). Werden nur nach deiner ausdrücklichen Einwilligung geladen. Rechtsgrundlage: Art. 6 Abs. 1 lit. a DSGVO i. V. m. § 25 Abs. 1 TTDSG.
  • Marketing (Opt-in): Werkzeuge für personalisierte Werbung und Conversion-Tracking (z. B. Meta Pixel, Google Ads). Werden nur nach deiner ausdrücklichen Einwilligung geladen. Rechtsgrundlage: Art. 6 Abs. 1 lit. a DSGVO i. V. m. § 25 Abs. 1 TTDSG.

Geplante Tools

Sobald die folgenden Werkzeuge eingebunden werden, informieren wir hier über die konkreten Verarbeitungsvorgänge. Bis dahin werden keine entsprechenden Skripte geladen, auch wenn du im Banner zugestimmt hast:

  • Google Analytics 4 (Statistik)
  • Google Ads Conversion Tracking (Marketing)
  • Meta Pixel / Facebook Pixel (Marketing)
  • Newsletter-Tool für E-Mail-Anmeldungen (z. B. Brevo, Mailchimp — Anbieter wird hier ergänzt sobald entschieden)

Widerruf deiner Einwilligung

Du kannst deine Einwilligung jederzeit widerrufen oder anpassen. Klicke dafür auf den Link „Cookie-Einstellungen" im Footer dieser Seite. Der Widerruf gilt für die Zukunft.

Speicherdauer

Deine Einwilligungs-Entscheidung wird für 12 Monate in deinem lokalen Browser-Speicher (localStorage) gespeichert. Danach erscheint der Banner erneut, damit du deine Auswahl bestätigen oder ändern kannst.
